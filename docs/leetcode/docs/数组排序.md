# 数组排序
记录常见的js排序方法

## 冒泡排序

```js
function bubbleSort(arr) {
  if(arr.length <= 1) return arr;
  for (let i = 0; i < arr.length; i++) {
    // 提前退出标志位
    let flag = false;
    for (let j = 0; j < arr.length - i - 1; j++) {
      if(arr[j] > arr[j+1]) {
        let tmp = arr[j];
        arr[j] = arr[j+1];
        arr[j+1] = tmp;
        // 如果有数据交换就设置为true，如果无数据交换，说明数组已经是有序的了
        flag = true;
      }
    }
    if(!flag) break;
  } 
  return arr;
}
```
两次循环，时间复杂度O(n);
